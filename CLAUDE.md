Проект, в котором создается guix-манифест для приложения telegram-desktop.

Если надо посмотреть исходный код telegram-desktop, ты можешь его найти в /home/evgenii/guix/telegram-desktop/sources
(например, если надо посмотреть какие сабмодули используются). В sources лежат просто исходники для примера. Сборка в
guix происходит не из них. Если ты хочешь внести какие-то модификации в исходники, ты должен делать это с помощью
модифицирования манифеста (добавление новых пакетов, создание патчей и т.д.)

Если нужны исходники guix-канала, смотри в /home/evgenii/guix/guix. Пример манифеста telegram-desktop можно найти в
/home/evgenii/guix/guix/gnu/packages/telegram.scm.

Наша цель - создать полнофункциональный манифест, который ПОЛНОСТЬЮ и ДО КОНЦА собирает приложение и позволяет его
запустить. Definition of done: 1) пакет полностью собирается, 2) бинарный файл запускается без ошибок.

При необходимости, ищи информацию в интернете.

Команду guix build следует звать с ключами -K -L /home/evgenii/guix/telegram-desktop -v 3 -c 16

Полная сборка telegram-desktop занимает около часа. Учти это, когда выставляешь таймауты.

При обновлении версии также обновляй версию в README.md (в примере commit). После успешной сборки создавай коммит и
выставляй на него тэг вида `v<версия>` (например, `v6.6.2`).
